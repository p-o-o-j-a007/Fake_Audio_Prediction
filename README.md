# Fake_Audio_Prediction
# 🎙️ Fake Audio Prediction using Machine Learning

## 📌 Project Overview

This project aims to detect whether an audio clip is **Real** or **Fake** using Machine Learning techniques. Audio features are extracted using MFCC (Mel-Frequency Cepstral Coefficients) and used to train a classification model.

---

## 🚀 Features

- Audio preprocessing
- MFCC feature extraction
- Machine Learning model training
- Real/Fake audio prediction
- Performance evaluation

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Librosa
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📂 Project Files

- Final_Project.ipynb
- Real Audio.zip
- fake_audio_files.zip

---

## 🔄 Workflow

1. Load Dataset
2. Audio Preprocessing
3. Feature Extraction (MFCC)
4. Train/Test Split
5. Model Training
6. Prediction
7. Model Evaluation

---

## 📊 Output

The trained Machine Learning model predicts whether an audio sample is **Real** or **Fake** with high accuracy.
